# Plan 49 ‚Äî Unify Conn naming across `time/` and `hifi/`

## Goal

Bring `time/` and `hifi/` Conn-constant names into a single coherent
naming scheme, fixing four classes of drift that crept in as the two
modules grew separately.

## Context

Sprints 1-6 grew `time/` and `hifi/` independently. Each module
followed `CLAUDE.md`'s 8-character convention but the mechanical rules
for picking 4-char *side codes* drifted apart. Audit (one-shot) found:

1. **Duration trio** uses three different rules for the same family
   ‚Äî `DURN` (no crate tag), `STDR` (crate prefix), `HDUR` (crate
   prefix + `DUR` suffix). Only `HDUR` reads cleanly.
2. **`EUTC*`** mixes two reference epochs (J1900 UTC for
   `EUTCHDUR`, UNIX UTC for `EUTCNANO` / `EUTCF064`) under one
   prefix. The name should disclose which zero is in force.
3. **All nine `E*F064` Conns** in `hifi/epoch.rs` have the float on
   the **wrong side of the name** ‚Äî the type is `Conn<F064,
   Extended<Epoch>>` but the name has F064 *trailing*, violating
   the canonical-example rule (`F064F032` = `Conn<F064, F032>`).
4. **`MONTNZ08`** uses an `AB12`-shape NonZero side code; the
   canonical 1L+3D form `N008` is established in `fixed/i8.rs:60`
   (`I008N008`). One-off divergence.

The detailed audit lives in the plan-mode workspace under
`~/.claude/plans/` for historical reference.

### T5 ‚Äî Add lint script `scripts/check_conn_names.py`

Source-of-truth side-code registry, parser for
`pub NAME : L => R` macro sites + `pub const NAME: Conn<L, R>`
sites, asserts `NAME == sidecode(L) ++ sidecode(R)` and shape
conformance. Fails on first mismatch with file:line + suggested
correct name.

Registry shape (literal in script):
```python
SIDECODE = {
    "F064":              "F064",   # ExtendedFloat<f64>
    "F032":              "F032",
    "i8":                "I008",
    "i16":               "I016",
    # ... (full int / fixed / nonzero ladders)
    "Extended<Time>":    "TIME",
    "Extended<Date>":    "DATE",
    "Duration":          "TDUR",   # time::Duration
    "Extended<Duration>":"TDUR",
    "StdDuration":       "SDUR",
    "Extended<StdDuration>": "SDUR",
    "HD":                "HDUR",   # hifitime::Duration alias
    "Extended<HD>":      "HDUR",
    "PrimitiveDateTime": "PDTM",
    "Extended<OffsetDateTime>": "ODTM",
    "Extended<MonthName>": "MONT",
    "Extended<Weekday>":   "WKDY",
    # Epoch is multi-scale: handled specially via macro-name
    # (etai_*, eutc_*, eunx_*, egps_*, etc. ‚Üí ETAI/EUTC/...)
}
```

Script returns 0 (clean) / 1 (mismatch). Wire into hooks in T6.
