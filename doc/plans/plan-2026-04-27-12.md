### T2 ‚Äî Reorder README sections

Promote `What is a connection?` from h3 to h2. Add a new
`## How connections work` section walking through three
`Ordering ‚Üî bool` examples with verification tables, mirroring the
Haskell `connections` README's pedagogical progression:

1. one-sided `Conn::new_left` (analog of `ordbin :: Cast 'L`) ‚Äî 2-col table
2. flipped one-sided (analog of `binord`) ‚Äî 2-col table, motivates the chain
3. full triple `Conn::new` (analog of `Cast k`) ‚Äî 3-col table, with a closing
   note that `round`/`truncate`/`midpoint`/`interval`/`median` work on every
   `Conn<A, B>` regardless of construction; on one-sided ones they degenerate
   trivially because `floor == ceil`.

Add `## Installation` (cargo add, MSRV, `f16` / `testing` flags). Final
order: preamble ‚Üí What are Galois connections? ‚Üí How connections work ‚Üí
Why this crate ‚Üí Installation ‚Üí The core type ‚Üí Quick tour ‚Üí What's

‚Üí Testing ‚Üí Links.
